## Myi3
I3wm simple: urxvt, sublime text, vim, etc

## Dependencies
* Feh
* dmenu
* Awesome Font
* NetworkManager-applet
* i3-wm
* i3lock-color
* pulseaudio

## My vim config

My vim config is [here](https://github.com/ctnguyenvn/vimrc) or install with:

```
git clone https://github.com/ctnguyenvn/vimrc
cd vimrc && bash install.sh
```
